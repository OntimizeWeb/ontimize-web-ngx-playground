What is Ontimize Web?

Ontimize Web is web application framework based on Angular for building bussiness software. Ontimize Web makes use of Angular framework and its UI Component framework (Angular Material) to provide a set of reusable, well-teste and accesible components apart from a number of standard services and functionalities.
